The special @lib keyword is a configurable, relative or absolute, URL path that can be used as a shortcut to load components from a shared location. By default it points to the zKit website that is a collections of sample components made with zUIx.

Transform and reshape

Keep HTML content code simple and clean and then transform it by applying templates or loading components on it.
With automatic fields mapping (data-binding) or JavaScript, templates and components can smoothly transform a minimal set of HTML elements into a rich and interactive content.

Optimize speed and performance

Either you're writing a Progressive Web App (PWA) or a simple website, lazy loading is a built-in feature that can boost up site speed by loading content and components only when they come visible in the screen viewport.

The following code loads the view_pager component (from zKit) and adds 30 random pictures using the example random_image_item component.

ViewPager with lazy-loaded items ```html
{% raw %}{% set list = Array(30) %} {% for x in list %}
{% endfor %}{% endraw %}
```

The result of this snippet is the image strip shown below.

{% endfor %}

Drag or fast swipe over it fast_rewind horizontally fast_forward to actually see the lazy-loading in action.

More features

Just to name some, controllers overlapping and hot view swapping are unique zUIx features. The first one allows loading of multiple controllers on the same view element. The second one allow to dynamically replace a view with a different (or updated) template at any time.

zUIx is a lightweight library (13kB gzipped), though still offering a full feature set.

  • Events, Behaviors and Hooks
  • Components and fields caching
  • Automatic events unbinding
  • Integrated resources loader
  • Integrated jQuery-alike DOM helper
  • Integrated app-bundler
zUIx Web Starter

Is an integrated solution for building and publishing any kind of web project, from a simple website or blog to more complex web applications.

  • Built-in web server and build scripts
  • Automatic generation of zUIx components bundle
  • Compiles LESS CSS style files
  • Integrates ESLint check and error reporting, suitable for CI tools
  • Ships with static-site engine and its fully featured commands set for creating static pages (Front Matter, Swig, Markdown, ...).
  • Boost up site loading with automatic generation of PWA scripts (WIP)
